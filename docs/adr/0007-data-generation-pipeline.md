# 0007: Data Generation Pipeline

## Status

Accepted

## Context

The mandatory ADR list includes Mode B data pipeline decisions, but Cinematheca is Mode A.

## Decision

There is no offline data-generation backend in v1. Static metadata is generated by the frontend build only.

## Consequences

`make data` is intentionally a no-op that documents Mode A. No Parquet, SQLite, scheduled scraping, or GitHub Release data artifact exists in v1.

## Alternatives Considered

A Mode B preset pipeline was rejected because current presets are small, deterministic TypeScript constants.
